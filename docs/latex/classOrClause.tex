\subsection{Detailed Description}
Class encompassing a list of S\-A\-T variables (type {\ttfamily \hyperlink{classVariable}{Variable}}) linked by or operators. 

For example, the or clause \char`\"{}var1 O\-R var2 O\-R var3 O\-R not var 4 O\-R var 5\char`\"{} can be defined as Or\-Clause(5, var1, var2, var3, not var4, var5) \begin{DoxyAuthor}{Author}
